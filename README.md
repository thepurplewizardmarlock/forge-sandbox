# Forge Sandbox

A beginner-friendly Python sandbox for learning by doing.

## What's here

- `hello.py` — starter script: a simple greeting function with input validation
- `test_hello.py` — tests that verify `hello.py` works correctly

## How to run

```bash
# Run the script
python3 hello.py

# Run the tests
python3 -m pytest test_hello.py -v
# or without pytest:
python3 test_hello.py
```
